# 一键部署

## 🚂 Railway (推荐)
[\![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new?template=https://github.com/wanwanzajiang/aotu-crm)

点击按钮 → GitHub授权 → 选择仓库 → 自动部署（约2分钟）

## 🎯 Render
[\![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/wanwanzajiang/aotu-crm)

点击按钮 → 创建账户 → 连接GitHub → 自动部署

## 💻 本地运行
```bash
cd backend
npm install
node server.js
# 访问 http://localhost:3456
```
